# SunnyLand - Introdução

SunnyLand é um projeto desenvolvido para a disciplina de Programação à Jogos do curso de Jogos Digitais, da Universidade do Estado da Bahia - Uneb. O projeto tem como finalidade aplicar o conhecimento em sala de aula, cruiando assim um simples jogo utilizando recursos da Unity. Trata-se de um jogo 2D. O nome do projeto original se chamava "A invasão de SunnyLand pelas 5 bestas do Apocalipse", entretanto, os desenvolvedores optaram em deixar só "SunnyLand. O Objetivo principal desse projeto é aplicar o conhecimento da linguagem de programação C# voltado para games.
